How It Works

Clean flow, no creepy shortcuts.

This experience is intentionally built around explicit participation from the device owner.

1 Open the page on the device being shared.
The device owner enters a name, trusted viewers, and confirms consent.
2 Choose how precise the shared coordinates should be.
Exact, nearby, or area mode changes the exported coordinate precision.
3 Start the browser GPS session.
HaloShare shows the browser-reported accuracy radius and timestamp instead of inventing an address.
4 Copy a snapshot link for a trusted viewer.
The receiving side can paste the link back into HaloShare to inspect the shared snapshot.

Accuracy Policy

What this page does and does not claim.

Accuracy depends on the device, browser, permission state, GPS visibility, and network conditions. This page is designed to be honest about that signal.

  • It will not track someone from a phone number, contact list, or silent lookup.
  • It will not invent a street name, building, or place when the browser only provides coordinates.
  • It shows freshness and accuracy radius so you can judge confidence instead of trusting fake certainty.
